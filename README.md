# Установка

1. Скачать директорию.
2. Создать среду
```
python -m venv venv
```
4. Активировать среду `./venv/Scripts/activate`
5.
```
pip install -r requirements.txt
```

# Запуск

1.
```
py ./run.py
```
3. Потыкать эндпоинты можно по ссылке [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)

# Тестирование

1.
```
pytest
```
